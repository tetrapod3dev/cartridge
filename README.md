## 소개
Cartridge는 *The Binding of Isaac*의 모드 조합(프리셋)을 손쉽게 만들고 전환할 수 있는 Windows 데스크톱 앱입니다. 세이브/설치/옵션 폴더를 자동 탐지하고, 자주 쓰는 경로를 빠르게 열 수 있습니다.

## 주요 기능
- **홈 화면**: Isaac 관련 주요 폴더(세이브/설치/옵션) 자동 탐지 및 바로가기 제공
- **인스턴스/프리셋 관리**: 다양한 모드 조합을 저장/불러오기
- **컨텐츠 / 슬롯 머신**: 방송/콘텐츠용 미니 툴
